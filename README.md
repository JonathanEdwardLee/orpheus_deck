# ORPHEUS DECK

### Junkfeathers Tech Four-Track Recorder

## Four Tracks. One Tape. No Excuses For the Muses.

*A digital cassette-style four-track recorder built for capturing ideas before they disappear.*

---

## What Is Orpheus Deck?

Orpheus Deck is not a modern DAW.

It is a deliberately limited four-track cassette recorder inspired by:
- Tascam PortaStudios
- cassette overdubbing
- tape bouncing
- field recorders
- cheap cassette experiments
- hardware workflows
- lo-fi accidents
- late-night song ideas

The goal is not perfection.

The goal is:
- fast idea capture
- musical experimentation
- limitation-driven creativity
- preserving the feeling of analog tape workflows inside a modern Android device

---

## Philosophy

Modern recording software can become endless:
- endless tracks
- endless editing
- endless tweaking
- endless unfinished songs

Orpheus Deck intentionally pushes in the opposite direction.

You get:
- four tracks
- one virtual tape
- simple transport controls
- minimal editing
- commitment-based recording
- cassette-style limitations

The limitations are not a weakness.

They are the point.

Sometimes constraints create better art.

## Recording Philosophy

Orpheus Deck supports two recording approaches:

### CLEAN OVERDUB
Use headphones and layer tracks traditionally.

### LO-FI BLEED MODE
Let the phone speaker bleed back into the microphone intentionally.

Each generation can capture:
- room sound
- previous layers
- distortion
- noise
- accidental textures
- speaker bleed
- cassette-like buildup

This is inspired by old cassette-to-cassette overdubbing and primitive tape bouncing workflows.

Sometimes the artifacts become part of the song.
